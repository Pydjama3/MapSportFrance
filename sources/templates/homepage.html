Géograph - Tableau de configuration

Edition Jeux Olympiques 2024

 Bienvenue sur l'interface de configuration de GEOgraph, un outil "dernier cri" qui révolutionnera sans doute le monde des statistiques en France (légère ironie): cette application vous permet de visualier des donnnées sur la carte de la France !
Pour cela, vous devez suivre le étapes suivantes:

  1. Placer les fichiers (.csv) de données dans le dossier "sources/data".
  2. Ajouter les jeux de données que vous voulez afficher . Attention, il n'est pas possible d'afficher plusieurs données pour un même département; l'affichage de plusieurs données permet, par exemple, d'afficher les valeurs pour certains départments, dans une autre couleurs que les autres:
    1. Si vous voulez ajouter des données à partir d'un fichier, appuyez sur ".csv" en bas de la page.
    2. Si vous voulez calculer des valeurs à partir de d'autres jeux données, appuyez sur "f(x)="
  3. Configurer les champs qui vous sont proposés: légende (comment ces données seront légendées sur la carte), nom du fichier source, en-têtes (nom en haut des colonnes concernants les codes des départements ou les valeurs), etc...
  4. Donner un titre global à votre magnifique croquis
  5. Enfin, appyer sur "Générer !" pour voir le résultat ! (un peu de patience après, nos serveurs croulent sous la demande ^_^)
